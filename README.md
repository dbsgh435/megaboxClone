## 프로젝트 개요
메가박스(megabox.co.kr) 사이트를 클론코딩, 지금 시작합니다!

## 프로젝트 일정
2-21.9.6 ~ 2021.9.8(예정)

## 프로젝트 방향
* 원본 사이트의 소스를 재활용
* 전체적인 레이아웃과 기능구현에 포커스
* 오류없게 기간 내에 완성해보는 것

## 프로젝트 진행단계
* 깃험 세팅/로컬 세팅
* 메가박스 타겟팅 --> 분석
* 파일/폴더 생성 관련된 작업 진행!
* 작업소스는 메가박스에서 다운, URL, 색상 추출...
